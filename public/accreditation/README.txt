Official NABH and NABL accreditation mark files required

Place authorized certification marks from the clinic's accreditation portal here.
Do not use unofficial logos from web searches.

Required files:
- nabh-certification-mark.png (or .svg / .webp — update lib/data.ts if extension differs)
- nabl-accreditation-symbol.png (or .svg / .webp)

After adding official files, set available: true for each mark in ACCREDITATION_MARKS
in lib/data.ts. Preserve mark proportions, colours, wording, and accreditation numbers.
